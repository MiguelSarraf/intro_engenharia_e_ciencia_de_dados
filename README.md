# Introdução à Engenharia e Ciência de dados
Repositório para armazenar códigos de exemplo para as camadas de processos de engenheria e ciência de dados abordados no curso introdutório ao tema que estará disponível em breve na Hotmart/Udemy.

## Configurando ambiente

Este código está sendo desenvolvido e testado e uma máquina Ubuntu18.04. Prentendo criar rotinas para outros sistemas operacionais em breve, mas primeiro o desenvolvimento principal tem que ser finalizado.

Passo-a-passo de configuração do ambiente para executar:

```
make env
source intro_venv/bin/activate
source environment_variables.sh
make requirements
```
